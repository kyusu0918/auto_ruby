【自動ルビプラグイン auto_ruby ver1.0.0】

====================================================================================================
■ 概要
====================================================================================================
テキストにルビを付加するプラグインです。
最初に登録したルビを同じ文字に自動で割り当てるので、何度もルビを振る必要はありません。

■ ファイル
====================================================================================================
1.auto_ruby: ティラノスクリプト用プラグインフォルダ
2.readme.txt: 当ファイル

====================================================================================================
■ 機能
====================================================================================================
1.テキストへのルビの表示
2.ルビの登録
3.登録済みのルビを使用したルビの表示
4.登録済みのルビの削除
5.登録済みのルビのクリア

====================================================================================================
■ 使い方
====================================================================================================
1.data/others/plugin にauto_ruby{バージョン}.zipファイル内の「auto_ruby」フォルダをコピーてください。
  ※下記構成で4ファイルがあるか確認してください。
  data/others/plugin/auto_ruby/init.ks
  data/others/plugin/auto_ruby/auto_ruby.js
  data/others/plugin/auto_ruby/auto_ruby.css
  data/others/plugin/auto_ruby/auto_ruby.json
2.first.ksからプラグインを呼び出してください。
  ※ティラノビルダーの場合は、data/scenario/system/plugin.ksからプラグインを呼び出してください。
  [plugin name=auto_ruby]
3.後述のタグリファレンスを参考にしてルビを使用します。

====================================================================================================
■ タグリファレンス
====================================================================================================

; 下記リファレンスの1～5のサンプルを、ティラノスクリプトのシナリオファイル（.ks）に、そのまま貼り付けても動作します。

;----------------------------------------------------------------------------------------------------
; 1.ルビ開始終了タグ
;
; 開始タグと終了タグで、ルビを表示したいテキストを囲みます。
;
; 【タグ】
; [srb] 開始タグ（StartRubyの略）
; [erb] 終了タグ（EndtRubyの略）
;
; 【パラメータ】
; text            省略可能  表示するルビを設定します。省略時は登録済みのルビを検索してルビを表示します。
;                           ※未登録の場合は、表示されません。
;
;----------------------------------------------------------------------------------------------------
;以下サンプル
;自動ルビプラグインでのルビの振り方
[srb text="じどう"]自動[erb]ルビプラグインでは、[srb text="かんたん"]簡単[erb]に[srb text="たんご"]単語[erb]をまとめて[srb text="せってい"]設定[erb]できます。[p]

;ルビを事前に登録
[arb ruby="誰" text="だれ"]
[arb ruby="帰" text="かえ"]

;登録済みの文字であれば、ルビを入れたい文字列をタグで囲むだけ
;個別にルビの設定をする必要はありません
[srb]誰[erb]もいねぇじゃねぇか。[p]
……[p]
[srb]帰[erb]るか。。。[p]

[font  size="30"   ]
#?
ちょっとまったーーーーー[p]
[resetfont  ]

#
[srb]誰だ[erb]！？[p]


;----------------------------------------------------------------------------------------------------
; 2.ルビ登録
;
; ルビストアにルビとテキストを登録します。
; ルビストアは、システム変数（sf）内にauto_ruby_store配列として作成されます。
;
; 【タグ】
; [arb]登録タグ（AddtRubyの略）
;
; 【パラメータ】
; ruby  必須      ルビをを振る文字列 ※同一の文字列を設定した場合は上書きされます
; text  省略可能  ルビの内容
;----------------------------------------------------------------------------------------------------
;以下サンプル
;ルビを事前に登録
[arb ruby="誰" text="だれ"]
[arb ruby="帰" text="かえ"]


;----------------------------------------------------------------------------------------------------
; 3.ルビ削除
;
; ルビストアに登録されたデータを削除します。
;
; 【タグ】
; [drb]削除タグ（DeleteRuby略）
;
; 【パラメータ】
; ruby  必須      ルビをを振る文字列
;----------------------------------------------------------------------------------------------------
;以下サンプル
;登録したルビを削除
[drb ruby="興味"]

[chara_mod  name="akane" face="happy"  ]

;ルビを削除したので表示されない
#あかね
わー。[srb]興味[erb]あるなんて、嬉しいなー。[p]


;----------------------------------------------------------------------------------------------------
; 4.ルビクリア
;
; ルビのストアデータを全てクリアします。
;
; 【タグ】
; [crb]クリアタグ（ClearRubyの略）
;
; 【パラメータ】
; なし
;----------------------------------------------------------------------------------------------------
;以下サンプル
;登録したルビを全て削除（クリア）
;ルビが表示されなくなる
[srb]横書[erb]きと[srb]縦書[erb]きをシーンによって使い分けることもできます[r][l]


;----------------------------------------------------------------------------------------------------
; 5.JSONファイルからのルビ登録
;
; JSONファイルからルビの一覧を読み込んで、ルビストアに登録します
;
; 【タグ】
; [arb_json]JSONファイルルビ登録タグ（AddRubyJSONの略）
;
; 【パラメータ】
; file  必須      JSONファイルへのパス 
;                 ※JSONファイルは「data\others\plugin\auto_ruby」のフォルダ内に作成してください
;                 ※サンプルとして「auto_ruby.json」をご利用ください
;----------------------------------------------------------------------------------------------------
;以下サンプル
;JSONファイルからルビを登録
[arb_json file="./data/others/plugin/auto_ruby/auto_ruby.json"]

;キャラクター登場
[chara_show  name="akane"  ]
#?

#あかね
もしかして、ノベルゲームの[srb]開発[erb]に[srb]興味[erb]があるの？[p]
